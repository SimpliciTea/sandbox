Processing.JS exercise: Fishtank

This is my implementation of a KhanAcademy programming project, which can be found here.

It is an example of an HTML5 canvas element being drawn on using the Processing.JS programming language. The advent of the Canvas element in HTML5 brought graphics natively to the web, and so Processing was migrated to be used as a JavaScript library. It's all pretty straightforward once you get the hang of it!

Features:

  • Click to draw randomly generated fish! (size of body, eyes, tail, color of body, tail)
  • Randomly generated seawead using Bezier curves originating from base of canvas frame (try refreshing!)

Challenges:

  • Does not render on htmlpreview/rawgit! You have to click the link to see it on Khan Academy unless you're running the page on a simple web server like python's http.server -- in which case the sketch shows at right.
  • Figuring out how Bezier curves and their control points work!
  • Selecting appropriate ranges for Bezier fn points to draw convincing seaweed
  • Figuring out how to embed the canvas in my own page with Processing.JS
  • ...which required setting up my first local webserver using Python!!
